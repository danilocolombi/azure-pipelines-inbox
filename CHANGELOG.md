# Changelog

## 0.1.0

- Initial release.
- Runs inbox tree across subscribed projects with live status icons.
- Live timeline (Stage → Job → Task) that updates while a run is in progress.
- Live log tail in a webview panel with `##[error]`/`##[warning]`/`##[section]` highlighting.
- Filters: only my runs, status, branch.
- Opt-in run actions: cancel and re-run (gated behind a Build Read & Execute PAT).
